
# 🎯 AI ATS Resume Optimizer
A high-performance resume analysis tool built with **Streamlit** and **Groq (Llama 3)**.

## ✨ Features
- **Instant Scoring:** Get a 0-100% match score.
- **Critical Gaps:** Identifies missing mandatory skills/experience.
- **Optimization Tips:** Suggests what to add and what to remove (fluff).
- **Professional Dashboard:** Visual breakdown of resume performance.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **LLM:** Groq API (Llama 3.3 70B)
- **Parsing:** PyPDF2, python-docx

## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/codeitwithaayush/AI-ATS-Resume-Optimizer.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run main.py`
4. Enter your Groq API Key in the sidebar.
